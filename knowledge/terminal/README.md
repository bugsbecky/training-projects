# Terminal toolkit

Commands you actually use while learning. Prefer real examples from your machine.

## Everyday

| Command | What it does (simple) | When I use it |
|---|---|---|
| `pwd` | Shows “where am I?” (current folder) | before `cd` / when lost |
| `ls` | Lists files in this folder | see what is here |
| `cd folder` | Moves into a folder | go to project / Java folder |
| `cd ..` | Goes up one folder | back out |

## Learning Java (fill as you go)

| Command | What it does | Notes |
|---|---|---|
| `javac File.java` | Compiles Java source | creates `.class` |
| `java ClassName` | Runs the compiled program | no `.java` / usually no `.class` in the name |

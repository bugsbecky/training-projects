# Concepts

One markdown file per idea you understand well enough to explain in your own words.

## Template for a new file

Copy this into `concepts/your-topic.md`:

```markdown
# Concept name

**In one sentence:** …

**Analogy:** …

**I can show it by:** (terminal command or tiny example idea — not full homework)

**Related:** links to open-questions or mistakes files

**Date learned:** YYYY-MM-DD
```

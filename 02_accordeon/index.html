Was ist die DOM

DOM steht für Document-Object-Model. Sobald die URL aufegrufen wird, sendet das Backend das HTML-Document an den Browser, welcher daraus die DOM entwickelt. Diese speichert die Elemente der Webseite als Objekte lokal auf der client-side. Diese können durch JavaScript beeinflusst werden.

Was ist DOM-Manipulation und wofür brauche ich das?

DOM-Manipulation verändert DOM-Objekte per JavaScript, um Webseiten interaktiv zu machen (z. B. Dark Mode). Häufig verwendet man das, um neue Klassen via element.classList.add("name") hinzuzufügen. Damit diese neuen Klassen den alten Stil überschreiben, müssen die neuen CSS-Regeln weiter unten stehen (Kaskade) oder eine höhere Spezifität besitzen.
